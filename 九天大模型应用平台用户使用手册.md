# 4.1.1 鉴权

LLM Studio API 使用 token 进行身份验证，所有 API 请求都应在 Authorization HTTP 头中包含您的 token 密钥如下所示：

Authorization: Bearer token

token 由用户订单的 APIKEY 和有效秒数生成，为确保防止篡改，每个用户、每次生成的签名密钥均不同，例如：

每个订单的 APIKEY 由密钥 id 与签名密钥组成，例如：
